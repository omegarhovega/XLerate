Excel Add-in

XLerate

XLerate is an Excel add-in for financial modeling workflows, with fast ribbon tools for tracing formulas, checking horizontal consistency, cycling through model formats, filling formulas, and managing workbook-specific presets.

Download Manifest View GitHub Repository
If you used the older VBA-based XLerate, this site hosts the newer Office.js version. The VBA version still exists in the repository for reference, but the Office.js add-in is the current version.

Install

Download

Get the production add-in manifest:

manifest.xml shortcuts.json

Add to Excel

  1. Download manifest.xml.
  2. Open Excel Desktop.
  3. Go to Insert → My Add-ins.
  4. Use the custom add-in / upload manifest flow.
  5. Select the downloaded manifest.
  6. Open the XLerate ribbon tab.

Features

Modeling tools

  • Switch Sign
  • Smart Fill Right
  • Insert CAGR
  • Horizontal Check

Formatting tools

  • Cycle Number / Date / Cell / Text formats
  • Auto-color values and formulas
  • Workbook-specific presets
  • Settings side pane for customization

Trace tools

Trace Precedents and Trace Dependents can inspect formula relationships across sheets and through multiple levels, using a keyboard-friendly dialog.

Shortcuts

  • Ctrl+Shift+1 — Cycle Number Formats
  • Ctrl+Shift+2 — Cycle Cell Formats
  • Ctrl+Shift+3 — Cycle Date Formats
  • Ctrl+Shift+4 — Cycle Text Styles
  • Ctrl+Shift+R — Smart Fill Right
